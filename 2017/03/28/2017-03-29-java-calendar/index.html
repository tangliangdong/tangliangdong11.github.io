  • 加减分钟： field: 12

    • Calendar.MINUTE

  • 加减秒： field: 13

    • Calendar.SECOND

Calendar.AM_PM field: 9; 转换成12小时制

Calendar常量 基本上符合这样的规律：

  • Calendar.WEEK… 表示加减 周。
  • Calendar.DAY… 表示加减 日。
  • Calendar.HOUR… 表示加减小时。

除了 Calendar.DAY_OF_WEEK_IN_MONTH 这个比较特殊，是表示加减 周。
