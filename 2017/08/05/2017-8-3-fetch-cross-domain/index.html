fetch在旧版本的浏览器上兼容性不佳，特别是IE系的浏览器。但是现在有第三方插件可以很好的提升浏览器的兼容性。

因为浏览器的同源政策，这里也同样遇到了跨域的问题，之前是通过jsonp来解决的，这次通过设置服务器的CORS（跨域资源共享）来解决跨域问题。
